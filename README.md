# My PDF Image

A powerful online PDF & image processing platform built with [Next.js](https://nextjs.org).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- 30+ PDF tools (merge, split, compress, rotate, organize, watermark, page numbers, etc.)
- AI-powered tools (summarize, translate, contract review, invoice extraction, etc.)
- Image utilities (resize, compress, KB converter, etc.)
- OCR for scanned documents (English, Arabic, Urdu)
- Fully client-side processing for privacy and security
